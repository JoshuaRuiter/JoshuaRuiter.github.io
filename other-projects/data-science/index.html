Harmonic Features for Song Recommendations

Fall 2025 Machine Learning Bootcamp, The Erdős Institute

A project utilizing the Chordonomicon dataset, attempting to classify song features like genre, decade, and popularity based on features like unique chord density, major/minor chord frequency, and the presence of chord n-grams. Coding was done in Python with Numpy, Pandas, and Scikit-Learn.
